# 🧺 스마트 빨래 날씨 감지 시스템 (Smart Laundry Weather System)

Raspberry Pi 5 기반의 **스마트 빨래 날씨 감지 시스템**입니다.
날씨(맑음/비)를 감지하여 자동으로 빨래를 보호하는 임베디드 시스템 프로젝트입니다.

---

## 📖 프로젝트 소개

비가 오는 것을 감지하지 못해 널어놓은 빨래가 젖는 불편함을 해결하기 위한 프로젝트입니다.
Raspberry Pi 5와 각종 센서를 활용하여 **비를 감지하면 자동으로 빨래(차양/지붕)를 안으로 들이거나 덮어** 빨래가 젖지 않도록 보호합니다.

집 모형을 제작하여 실제 동작을 시연할 수 있도록 구성하였으며, 센서 데이터에 따라 모터가 동작하는 구조입니다.

### 주요 기능
- 🌧️ **빗물 감지**: 빗물 센서로 비가 오는지 실시간 감지
- ☀️ **자동 복구**: 비가 그치면 다시 빨래를 밖으로 내보냄
- ⚙️ **모터 제어**: 서보/DC 모터로 차양 또는 빨래 건조대 자동 제어
- 💡 **상태 표시**: LED 및 부저로 현재 상태 알림

---

## 🔄 동작 시나리오

### ☀️ 맑을 때 (정상 상태)
1. 빗물 센서가 물기를 감지하지 않음
2. 빨래 건조대(차양)가 **밖으로 펼쳐진 상태** 유지
3. 초록색 LED 점등 → "건조 중" 상태 표시
4. 빨래가 햇빛에 정상적으로 건조됨

### 🌧️ 비 올 때 (보호 상태)
1. 빗물 센서가 물기를 감지
2. 모터가 작동하여 빨래 건조대를 **안으로 들이거나 지붕으로 덮음**
3. 빨간색 LED 점등 + 부저 알림 → "비 감지! 빨래 보호 중" 상태 표시
4. 비가 그치면 다시 맑을 때 상태로 자동 복귀

## 🛠️ 사용 부품 목록

| 부품 | 용도 |
|------|------|
| Raspberry Pi 5 | 메인 컨트롤러 (제어 보드) |
| 빗물 감지 센서 (Rain Sensor) | 비/물기 감지 |
| 서보 모터 (SG90) 또는 DC 모터 | 차양/건조대 구동 |
| 모터 드라이버 (선택) | DC 모터 사용 시 전류 제어 |
| LED (빨강/초록) | 상태 표시 |
| 부저 (Buzzer) | 비 감지 알림 |
| 브레드보드 & 점퍼선 | 회로 연결 |
| 저항 (220Ω 등) | LED 보호 |
| 집 모형 (하드보드/우드락) | 시연용 하우징 |

> 📌 상세 부품 사양은 [docs/parts.md](docs/parts.md)를 참고하세요.
