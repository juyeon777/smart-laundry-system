# 🧺 스마트 빨래 날씨 감지 시스템 (Smart Laundry Weather System)

Raspberry Pi 5 기반의 **스마트 빨래 날씨 감지 시스템**입니다.
날씨(맑음/비)를 감지하여 자동으로 빨래를 보호하는 임베디드 시스템 프로젝트입니다.

---

## 📖 프로젝트 소개

비가 오는 것을 감지하지 못해 널어놓은 빨래가 젖는 불편함을 해결하기 위한 프로젝트입니다.
Raspberry Pi 5와 각종 센서를 활용하여 **비를 감지하면 자동으로 빨래(차양/지붕)를 안으로 들이거나 덮어** 빨래가 젖지 않도록 보호합니다.

집 모형을 제작하여 실제 동작을 시연할 수 있도록 구성하였으며, 센서 데이터에 따라 모터가 동작하는 구조입니다.

### 주요 기능
- 🌧️ **빗물 감지**: 빗물 센서(MH-RD)로 비가 오는지 실시간 감지
- ☀️ **자동 복구**: 비가 그치면 다시 맑음 상태로 자동 복귀
- ⚙️ **모터 제어**: 서보모터 2개로 지붕(차양)과 해 모형을 자동 제어
- 🔘 **수동 제어**: 버튼 2개로 맑음/비 모드를 직접 전환 (수동 모드 중 비 감지 시 자동 오버라이드)
- 💡 **상태 표시**: LED(빨강=맑음 / 파랑=비)와 수동형 부저 멜로디로 현재 상태 알림

---

## 🔄 동작 시나리오

### ☀️ 맑을 때 (정상 상태)
1. 빗물 센서가 물기를 감지하지 않음
2. 지붕(차양)이 **접혀** 빨래가 노출되고, **해 모형이 올라감**
3. **빨간색 LED** 점등 + 부저 도미솔도 상행음 → "맑음" 상태 표시
4. 빨래가 햇빛에 정상적으로 건조됨

### 🌧️ 비 올 때 (보호 상태)
1. 빗물 센서가 물기를 감지 (DO = LOW)
2. 서보모터가 작동하여 지붕(차양)을 **펼쳐 빨래를 덮고**, **해 모형이 내려감**
3. **파란색 LED** 점등 + 부저 경고음 반복 → "비 감지! 빨래 보호 중" 상태 표시
4. 비가 그치면 다시 맑을 때 상태로 자동 복귀

### 🔘 수동 제어
1. **버튼1**을 누르면 맑음 모드, **버튼2**를 누르면 비 모드로 강제 전환
2. 수동 모드 중에도 빗물 센서가 비를 감지하면 **자동으로 비 모드로 오버라이드**되어 빨래를 보호

## 🛠️ 사용 부품 목록

| 부품 | 용도 |
|------|------|
| Raspberry Pi 5 | 메인 컨트롤러 (제어 보드) |
| 빗물 감지 센서 (MH-RD) | 비/물기 감지 |
| 서보 모터 (SG90) × 2 | 지붕(차양) / 해 모형 구동 |
| LED (빨강/파랑) | 상태 표시 (맑음/비) |
| 부저 (수동형 3핀) | 상태 알림음 (PWM 멜로디) |
| 택트 스위치(버튼) × 2 | 수동 맑음/비 제어 |
| OLED 128x64 (I2C) | 상태 표시 화면 — *보류, 재진행 예정* |
| 브레드보드 & 점퍼선 | 회로 연결 |
| 저항 (220Ω) × 2 | LED 보호 |
| 집 모형 (플라스틱 통 + 보드롱 등) | 시연용 하우징 |

> 📌 상세 부품 사양은 [docs/parts.md](docs/parts.md)를 참고하세요.

## 🚀 실행 방법

```bash
# 필수 라이브러리 설치 (Raspberry Pi 5)
sudo apt install python3-lgpio
pip3 install gpiozero

# Raspberry Pi 5에서 실행
cd smart-laundry-system
python3 src/main.py
```

> ℹ️ 라즈베리파이 5는 `RPi.GPIO`를 지원하지 않아 **lgpio**를 사용합니다. 서보 제어는 `gpiozero.Servo`를 사용합니다.

### 📌 GPIO 핀 배정 (BCM)

| 핀 | 부품 | 핀 | 부품 |
|----|------|----|------|
| GPIO 17 | 빗물 센서 DO | GPIO 24 | 버튼1 (수동 맑음) |
| GPIO 18 | 서보1 (지붕) | GPIO 25 | 버튼2 (수동 비) |
| GPIO 19 | 서보2 (해 모형) | GPIO 22 | 빨간 LED (맑음) |
| GPIO 27 | 부저 (능동형) | GPIO 23 | 파란 LED (비) |

> 자세한 배선은 [docs/wiring.md](docs/wiring.md) 참고.
