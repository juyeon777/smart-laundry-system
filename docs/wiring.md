# 🔌 회로 연결 설명

Raspberry Pi 5와 센서·서보모터·LED·부저·버튼을 연결하는 **회로 구성 가이드**입니다.

---

## 1. 개요

MH-RD 빗물 센서로 비를 감지하고, 그 결과에 따라 **서보모터 2개**(지붕 차양 / 해 모형)를 동작시키며,
**LED·부저**로 현재 상태를 표시합니다. **버튼 2개**로 맑음/비를 수동 제어할 수 있습니다.
모든 부품은 Raspberry Pi 5의 **GPIO 핀**에 연결됩니다.

> ⚠️ GPIO 핀 번호는 **BCM 기준**입니다. 실제 배선 시 핀 위치를 꼭 확인하세요.
> ℹ️ OLED·카메라는 최종적으로 **제외**되었습니다. (OLED는 테스트 결과 **모듈 불량**으로 확인) 상태 확인·원격 제어는 **웹 대시보드**(같은 와이파이에서 `http://<Pi-IP>:5000`)로 대체했습니다.

---

## 2. 핀 연결표

| 부품 | 연결 핀 (BCM) | 물리 핀 | 비고 |
|------|---------------|---------|------|
| MH-RD 빗물 센서 DO | GPIO 17 | 11 | 디지털 출력 (**LOW=비 감지**) |
| MH-RD 빗물 센서 VCC | 5V | 2 / 4 | 전원 |
| MH-RD 빗물 센서 GND | GND | 6 등 | 공통 접지 |
| 서보모터 1 (지붕 차양) 신호 | GPIO 18 (PWM) | 12 | 하드웨어 PWM 권장 |
| 서보모터 2 (해 모형) 신호 | GPIO 19 (PWM) | 35 | 하드웨어 PWM 권장 |
| 서보모터 1·2 VCC | 5V | 2 / 4 | 전원 (전류 주의) |
| 서보모터 1·2 GND | GND | 9 등 | 공통 접지 |
| 빨간 LED (+) — 맑음 표시 | GPIO 22 | 15 | 220Ω 저항 직렬 연결 |
| 파란 LED (+) — 비 표시 | GPIO 23 | 16 | 220Ω 저항 직렬 연결 |
| 부저(수동형) I/O | GPIO 27 | 13 | PWM으로 음 출력 |
| 부저 VCC | 5V | 2 / 4 | 전원 (3.3V도 가능, 소리 작음) |
| 부저 GND | GND | 14 등 | 공통 접지 |
| 수동 버튼 1 (맑음) | GPIO 24 | 18 | 내부 풀업, 누름=LOW |
| 수동 버튼 2 (비) | GPIO 25 | 22 | 내부 풀업, 누름=LOW |
| 모든 GND | GND | - | 공통 접지 |

> ℹ️ 부저는 **수동형(Passive) 3핀 모듈(GND·I/O·VCC)** 입니다. PWM 주파수로 음정을 만들어 멜로디를 출력합니다. (맑음=도미솔도 상행음, 비=경고음, 각 1회)

## 3. 회로 연결 다이어그램 (개념도)

```
   [MH-RD 빗물센서] ──DO──→ GPIO 17 ┐
                                     │
                          ┌──────────┴──────────┐
                          │   Raspberry Pi 5     │
                          │                      │
   GPIO 18 (PWM) ─────────┤── 서보1 → 지붕 차양 펼침
   GPIO 19 (PWM) ─────────┤── 서보2 → 해 모형 구동
                          │
   GPIO 22 ───────────────┤→ [빨간 LED(맑음)]──220Ω──GND
   GPIO 23 ───────────────┤→ [파란 LED(비)]──220Ω──GND
   GPIO 27 ───────────────┤→ [부저(수동형) I/O]  VCC→5V, GND→GND
                          │
   GPIO 24 ──[버튼1 맑음]──┤  (내부 풀업 → GND)
   GPIO 25 ──[버튼2 비]────┘  (내부 풀업 → GND)

   [웹 제어] 폰/맥북 브라우저 ─ 와이파이 ─→ Pi (http://<Pi-IP>:5000)
```

---

## 4. 연결 시 주의사항

- **공통 접지(GND)**: 모든 부품(센서·서보·LED·부저·버튼)의 GND는 Raspberry Pi의 GND와 연결되어야 합니다.
- **LED 저항**: LED가 타지 않도록 반드시 **220Ω** 저항을 직렬로 연결합니다. (빨강·파랑 각 1개씩, 총 2개)
- **서보모터 전원**: SG90 2개를 동시에 구동하면 순간 전류가 커질 수 있습니다. Pi의 5V로 부족하면 **외부 5V 전원**을 사용하고 GND는 Pi와 공통으로 묶어주세요.
- **PWM 핀**: 서보 제어는 하드웨어 PWM이 가능한 **GPIO 18, GPIO 19** 사용을 권장합니다. (코드는 `gpiozero.Servo` 사용)
- **빗물 센서 로직**: DO 출력이 **LOW일 때 비 감지**입니다. (`gpio_read == 0` → 비)
- **버튼**: 내부 풀업을 사용하므로 한쪽은 GPIO, 다른 한쪽은 GND에 연결합니다. (누르면 LOW)
- **수동형 부저**: 자체 발진기가 없어 **PWM 주파수**로 음정을 만들어 구동합니다. (`lgpio.tx_pwm`, 듀티 50%) 3핀 모듈은 I/O→GPIO 27, VCC→5V, GND→GND로 연결합니다.
- **GPIO 라이브러리**: 라즈베리파이 5는 RPi.GPIO 미지원 → **lgpio** 사용.
- **웹 제어**: 별도 배선 없음. Pi가 같은 와이파이에서 웹서버(Flask)를 띄워 폰/맥북 브라우저로 상태 확인·제어합니다. (`http://<Pi-IP>:5000`)
