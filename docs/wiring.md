# 🔌 회로 연결 설명

Raspberry Pi 5와 센서·서보모터·OLED·LED·부저를 연결하는 **회로 구성 가이드**입니다.

---

## 1. 개요

MH-RD 빗물 센서로 비를 감지하고, 그 결과에 따라 **서보모터 2개**(지붕 차양 / 해 모형)를 동작시키며,
**OLED·LED·부저**로 현재 상태를 표시합니다. 모든 부품은 Raspberry Pi 5의 **GPIO 핀**에 연결됩니다.

> ⚠️ GPIO 핀 번호는 **BCM 기준**입니다. 실제 배선 시 핀 위치를 꼭 확인하세요.

---

## 2. 핀 연결표

| 부품 | 연결 핀 (BCM) | 물리 핀 | 비고 |
|------|---------------|---------|------|
| MH-RD 빗물 센서 DO | GPIO 17 | 11 | 디지털 출력 (비 감지 신호) |
| MH-RD 빗물 센서 VCC | 5V | 2 / 4 | 전원 |
| MH-RD 빗물 센서 GND | GND | 6 등 | 공통 접지 |
| 서보모터 1 (지붕 차양) 신호 | GPIO 18 (PWM) | 12 | 하드웨어 PWM 권장 |
| 서보모터 2 (해 모형) 신호 | GPIO 13 (PWM) | 33 | 하드웨어 PWM 권장 |
| 서보모터 1·2 VCC | 5V | 2 / 4 | 전원 (전류 주의) |
| 서보모터 1·2 GND | GND | 9 등 | 공통 접지 |
| OLED (I2C) SDA | GPIO 2 (SDA1) | 3 | I2C 데이터 |
| OLED (I2C) SCL | GPIO 3 (SCL1) | 5 | I2C 클럭 |
| OLED VCC | 3.3V | 1 | 전원 |
| OLED GND | GND | 14 등 | 공통 접지 |
| 노란 LED (+) | GPIO 22 | 15 | 220Ω 저항 직렬 연결 |
| 파란 LED (+) | GPIO 23 | 16 | 220Ω 저항 직렬 연결 |
| 부저(수동형) 신호 | GPIO 24 | 18 | PWM으로 음 출력 |
| 부저 GND | GND | 20 등 | 공통 접지 |
| 모든 GND | GND | - | 공통 접지 |

> ℹ️ OLED는 보통 **I2C 주소 0x3C**를 사용합니다. (`i2cdetect -y 1`로 확인)

---

## 3. 회로 연결 다이어그램 (개념도)

```
   [MH-RD 빗물센서] ──DO──→ GPIO 17 ┐
                                     │
                          ┌──────────┴──────────┐
                          │   Raspberry Pi 5     │
                          │                      │
   [OLED 128x64] ─SDA──── GPIO 2                 │
              └────SCL──── GPIO 3                 │
                          │                       │
   GPIO 18 (PWM) ─────────┤── 서보1 → 지붕 차양 펼침
   GPIO 13 (PWM) ─────────┤── 서보2 → 해 모형 구동
                          │
   GPIO 22 ───────────────┤→ [노란 LED]──220Ω──GND
   GPIO 23 ───────────────┤→ [파란 LED]──220Ω──GND
   GPIO 24 ───────────────┘→ [부저(수동형)]──GND
```

---

## 4. 연결 시 주의사항

- **공통 접지(GND)**: 모든 부품(센서·서보·OLED·LED·부저)의 GND는 Raspberry Pi의 GND와 연결되어야 합니다.
- **LED 저항**: LED가 타지 않도록 반드시 **220Ω** 저항을 직렬로 연결합니다. (노랑·파랑 각 1개씩, 총 2개)
- **서보모터 전원**: SG90 2개를 동시에 구동하면 순간 전류가 커질 수 있습니다. Pi의 5V로 부족하면 **외부 5V 전원**을 사용하고 GND는 Pi와 공통으로 묶어주세요.
- **PWM 핀**: 서보 제어는 하드웨어 PWM이 가능한 **GPIO 18, GPIO 13** 사용을 권장합니다.
- **OLED I2C**: SDA→GPIO 2, SCL→GPIO 3 고정. `sudo raspi-config`에서 **I2C 활성화** 필요.
- **수동형 부저**: 능동형과 달리 스스로 소리를 못 내므로, PWM 신호로 주파수를 만들어 구동합니다.
