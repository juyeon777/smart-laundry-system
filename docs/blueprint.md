# 🏠 집 모형 제작 가이드

스마트 빨래 날씨 감지 시스템의 시연을 위한 **집 모형 제작 가이드**입니다.

---

## 1. 개요

빗물 센서와 서보모터의 동작을 직관적으로 보여주기 위해 **플라스틱 통(짱구 통)을 집 몸체**로 사용하고,
앞쪽에 **보드롱 마당(30×45cm)** 을 두어 빨랫줄 구역을 구성합니다.
지붕(마분지 가동부)이 서보모터로 펼쳐져 비가 올 때 빨래를 덮으며, 옆쪽 해 모형(색지)이 서보모터로 오르내려 맑음/비를 표현합니다.

---

## 2. 모형 구성 재료

| 부위 | 재료 | 비고 |
|------|------|------|
| 집 몸체 | **플라스틱 통(짱구 통)** | 내부에 라즈베리파이·회로 배치 |
| 마당 바닥 | **보드롱(폼보드) 30 × 45cm** | 빨랫줄 구역 |
| 지붕(가동부) | **마분지** | 서보1로 펼쳐지는 차양 |
| 해 모형 | **색지(노랑/주황)** | 서보2로 오르내림 |
| 뼈대 보강 | **나무젓가락** | 구조 지지 / 빨랫줄 지지대 |
| 빨래 모형 | **종이 옷** | 빨랫줄에 부착 |

---

## 3. 배치 개념도

```
              [빗물센서]            [해 모형(색지)]
                  │                   ▲ 서보2로
            ┌─────┴─────┐ ← 지붕      │ 오르내림
            │  마분지    │   (서보1)   ●
            │  가동부    │
        ┌───┴───────────┴───┐
        │   플라스틱 통       │  ← 집 몸체
        │  (Pi·회로 내장)     │     (LED·부저)
        └─────────┬──────────┘
                  │
   ┌──────────────┴───────────────┐
   │   보드롱 마당 (30 × 45cm)      │
   │   나무젓가락 빨랫줄 + 종이 옷   │
   └───────────────────────────────┘
```

---

## 4. 부품 설치 위치

| 부품 | 설치 위치 |
|------|-----------|
| 빗물 센서 (MH-RD) | **지붕 윗면** (비를 직접 받는 위치) |
| 서보모터 1 (지붕) | **지붕 경첩 옆** — 마분지 차양 펼침/접힘 |
| 서보모터 2 (해 모형) | **집 옆면 외벽** — 색지 해 모형 오르내림 |
| 빨간/파란 LED | **집 몸체(내부 또는 정면)** — 맑음/비 표시 |
| 부저 (능동형) | **집 몸체 외부** — 알림음 |
| 수동 버튼 × 2 | **집 몸체 정면/측면** — 맑음/비 수동 제어 |
| 라즈베리파이 5 · 회로 | **플라스틱 통 내부** |
| (보류) OLED | **집 정면** — 재진행 시 설치 예정 |

---

## 5. 제작 순서

1. 플라스틱 통(짱구 통)을 집 몸체로 준비하고 내부에 라즈베리파이·브레드보드 공간 확보
2. 보드롱(30×45cm)으로 마당 바닥 제작 후 집 몸체 앞에 연결
3. 마분지로 지붕(차양)을 만들고 경첩 구조 + **서보모터 1** 설치
4. 지붕 윗면에 **빗물 센서** 부착
5. 집 옆면 외벽에 **서보모터 2** + 색지 **해 모형** 설치
6. 집 몸체에 **LED(빨강/파랑)·부저·버튼** 배치 및 배선 정리
7. 마당에 나무젓가락 **빨랫줄 지지대** + **종이 옷** 설치
8. 나무젓가락으로 약한 부위 뼈대 보강 후 전체 고정 및 동작 테스트
