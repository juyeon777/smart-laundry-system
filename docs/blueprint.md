# 🏠 집 모형 전개도 설명

스마트 빨래 날씨 감지 시스템의 시연을 위한 **집 모형 제작 가이드**입니다.

---

## 1. 개요

빗물 센서와 서보모터의 동작을 직관적으로 보여주기 위해 **직각삼각형 단면 구조**의 집 모형을 제작합니다.
지붕이 서보모터로 빨랫줄 위까지 펼쳐져 비가 올 때 빨래를 덮는 구조이며, 빗변 방향이 개방되어 내부의 OLED·LED·창문을 볼 수 있습니다.

---

## 2. 기본 구조 및 치수

| 항목 | 내용 |
|------|------|
| 단면 형태 | 직각삼각형 |
| 집 크기 | 밑변 **15cm** × 높이 **12cm** × 깊이 **10cm** |
| 개방 방향 | 빗변(경사면) 방향 오픈 → 내부 OLED·LED·창문 노출 |
| 지붕 동작 | 서보모터로 빨랫줄 위까지 펼쳐지는(차양) 구조 |
| 마당 바닥 | **25cm × 10cm** (빨랫줄 구역) |

---

## 3. 단면 개념도

```
        ▲ 높이 12cm
        │\
        │ \   ← 빗변(오픈 방향): 내부 OLED/LED/창문 보임
 뒷벽   │  \
        │   \   ┌─ 지붕(가동): 서보모터로 펼쳐짐
        │    \ /
        └─────\──────────────────────┐
        밑변 15cm   │   마당 바닥 25 x 10cm  │
                    │   (빨랫줄 구역)         │
                    └────────────────────────┘
```

---

## 4. 각 면 목록 (전개도)

| 면 | 치수 | 특이사항 |
|----|------|----------|
| 바닥(밑변 면) | 15cm × 10cm(깊이) | 집 하단 / 내부 부품(OLED·LED) 배치 |
| 뒷벽(직각변, 수직면) | 12cm(높이) × 10cm(깊이) | 외부에 부저 부착, 내부 뒷벽은 창문 표현 |
| 빗변 면 | 경사면 (약 19cm) × 10cm | **오픈 구조** — 내부가 보이도록 개방 |
| 지붕(가동부) | 경사면에 부착, 빨랫줄까지 펼쳐짐 | 서보모터로 동작하는 차양 |
| 측면(좌/우 삼각형) × 2 | 밑변 15cm × 높이 12cm 직각삼각형 | 좌측 외벽에 해 모형(서보2) 설치 |
| 마당 바닥 | 25cm × 10cm | 빨랫줄 구역 / 나무젓가락 지지대 + 종이 옷 |

> 📐 빗변 길이는 √(15² + 12²) ≈ **19.2cm** 입니다.

---

## 5. 부품 설치 위치

| 부품 | 설치 위치 |
|------|-----------|
| 빗물 센서 (MH-RD) | **지붕 고정부 위** (지붕 윗면) |
| 부저 (수동형) | **뒷벽 외부** |
| 서보모터 1 | **지붕 경첩 옆** (지붕 펼침 구동) |
| 서보모터 2 | **집 왼쪽 외벽** (해 모형 구동) |
| OLED / LED (노랑·파랑) | **내부 바닥** |
| 창문 | **내부 뒷벽** — 색지로 표현 |

### 설치 위치 개념도

```
                    [빗물센서]
                        │
                  ┌─────┴─────┐ ← 지붕(가동)
   [서보2]        │\          
   해 모형 ──────▶│ \  ← 빗변 오픈   [서보1] 지붕 경첩 옆
  (왼쪽 외벽)     │  \
                  │   \
        [부저]    │창문│ ← 내부 뒷벽 색지
       (뒷벽 외부)│    │
                  └────┴────────────────────┐
                  [OLED/LED 내부 바닥]  마당  │
                                        빨랫줄 │
                                       구역    │
                                  └────────────┘
```

---

## 6. 제작 순서

1. 보드롱(5mm)을 각 면 치수에 맞춰 재단 (바닥·뒷벽·측면 2개·지붕)
2. 직각삼각형 단면이 되도록 벽체 조립 (빗변 방향은 개방 상태로)
3. 지붕에 경첩 구조를 만들고 **서보모터 1**을 경첩 옆에 설치
4. 지붕 고정부 위에 **빗물 센서** 부착
5. 뒷벽 외부에 **부저**, 내부 뒷벽에 색지로 **창문** 표현
6. 집 왼쪽 외벽에 **서보모터 2** + 색지(노랑/주황) **해 모형** 설치
7. 내부 바닥에 **OLED·LED** 배치 및 배선 정리
8. 마당 바닥(25×10cm) 제작 후 나무젓가락 **빨랫줄 지지대** + **종이 옷** 설치
9. 순간접착제로 전체 고정 후 동작 테스트
