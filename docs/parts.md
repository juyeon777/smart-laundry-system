# 🧩 부품 목록 (Parts List)

스마트 빨래 날씨 감지 시스템에 사용되는 **부품 상세 목록**입니다.
보유 중인 키트 부품, 추가로 준비할 부품, 그리고 모형 제작용 공작 재료로 구분하여 정리했습니다.

---

## 1. 키트 보유 부품

이미 키트에 포함되어 보유 중인 부품입니다.

| 부품명 | 사양 | 수량 | 용도 |
|--------|------|------|------|
| 라즈베리파이 5 | Raspberry Pi 5 | 1 | 메인 제어 보드 |
| SG90 서보모터 | 마이크로 서보 | 1 | 지붕(차양) 구동 |
| OLED 디스플레이 | 128x64 0.96inch, I2C | 1 | ~~상태 표시 화면~~ → **제외 (모듈 불량 확인, 웹 대시보드로 대체)** |
| 부저 모듈 | **수동형(Passive)** 3핀 (GND·I/O·VCC) | 1 | 상태 알림음 (PWM 멜로디) |
| 빨간 LED | 5mm | 1 | 맑음 상태 표시 |
| 파란 LED | 5mm | 1 | 비 상태 표시 |
| 택트 스위치(버튼) | 4핀 푸시버튼 | 2 | 수동 맑음/비 제어 |
| 저항 | 220Ω | 2 | LED 보호 |
| 점퍼케이블 | M-M / M-F 등 | 다수 | 회로 연결 |
| 브레드보드 | 일반 사이즈 | 1 | 회로 구성 |

> ℹ️ MH-RD 빗물 감지 센서는 키트에 미포함 → 추가 구매 예정 (아래 참고)

## 4. 부품 수량 요약

| 구분 | 주요 항목 |
|------|-----------|
| 제어 보드 | 라즈베리파이 5 × 1 |
| 서보모터 | SG90 × 2 (키트 1 + 추가 1) |
| 센서 | MH-RD 빗물 감지 센서 × 1 |
| 입력 | 택트 스위치(버튼) × 2 (수동 맑음/비) |
| 표시/알림 | 부저(수동형) × 1, LED × 2(빨강/파랑) · *OLED는 불량으로 제외* |
| 원격 제어 | 웹 대시보드 (Flask, 폰/맥북 브라우저) — 추가 하드웨어 없음 |
| 회로 부자재 | 브레드보드 × 1, 미니 브레드보드 × 1, 저항 220Ω × 2, 점퍼케이블 다수 |
